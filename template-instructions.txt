
#  How to create a .docx template

1. Open Microsoft Word or LibreOffice
2. Create a new document
3. Add text with docx-templates commands:

Template example (one character):
---
Ð¡haracter Report

Name: +++INS character.name+++
Height: +++INS character.height+++
Mass: +++INS character.mass+++
Hair Color: +++INS character.hair_color+++
Skin Color: +++INS character.skin_color+++
Eye Color: +++INS character.eye_color+++
Birth Year: +++INS character.birth_year+++
Gender: +++INS character.gender+++
Homeworld: +++LINK ({url: character.homeworld})+++

Films:
+++FOR film IN character.films+++
  +++LINK ({url: $film})+++
+++END-FOR film+++

Vehicles:
+++FOR vehicle IN character.vehicles+++
  +++LINK ({url: $vehicle})+++
+++END-FOR vehicle+++

Ships:
+++FOR ship IN character.starships+++
  +++LINK ({url: $ship})+++
+++END-FOR ship+++

Created: +++INS new Date(character.created).toLocaleString("en-GB", { dateStyle: "long", timeStyle: "short" })+++
Edited: Created: +++INS new Date(character.edited).toLocaleString("en-GB", { dateStyle: "long", timeStyle: "short" })+++
URL: +++LINK ({url: character.url})+++

Report generated on: +++INS Intl.DateTimeFormat('en-US', { dateStyle: 'long' }).format(new Date())+++
---

4. Save as template.docx
5. Run: npm start (or node simple-example.js)
